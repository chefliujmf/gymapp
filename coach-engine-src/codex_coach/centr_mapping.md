# Centr Mapping

This file maps coach-prescribed gym work to Centr-friendly execution.

## Role

The coach defines the session.
Centr is used as source material and execution support when it is convenient.

The coach-owned exercise library is the reusable layer:
- [exercise_library.md](</Users/jmfiset/dev/cyclingcoach/codex_coach/exercise_library.md>)
- [source_library_workflow.md](</Users/jmfiset/dev/cyclingcoach/codex_coach/source_library_workflow.md>)

Centr can be used for:
- exercise demonstrations
- yoga, Pilates, mobility, and recovery sessions
- meal and snack ideas
- self-guided gym workouts when they match the planned movement pattern

When a Centr link or book-derived exercise idea becomes useful, capture it in the coach-owned library instead of relying on the external source alone.

Centr should not override:
- cycling load
- fatigue management
- weather-driven ride decisions
- the transition week after a major block
- the no-indoor-trainer summer rule

## Session templates

### Transition Strength And Mobility

Target session length:
- `60 min`, with `45 min` minimum and up to `75 min` if extra time is spent on easy mobility, breathing, or light trunk work

Movement slots:
- squat pattern
- hinge pattern
- horizontal pull
- single-leg support
- anti-rotation core
- mobility or yoga

Suggested exercise choices:
- hack squat, leg press, or goblet squat
- Romanian deadlift
- chest-supported row or seated cable row
- step-up or split squat
- Pallof press
- hip flexor, hamstring, calf, and t-spine mobility

Recommended Centr link:
- Centr Fusion: PowerFlow: `https://centr.com/workout/overview/0/13740/fusion-powerflow-move-to-release`

If that link does not fit the day, prescribe the coach mobility table directly. Do not give generic Centr search keywords.

Avoid during transition week:
- HIIT
- HIRT
- hard conditioning
- heavy leg day
- workouts that create next-day soreness

### Gym Strength Maintenance

Target session length:
- `60 min`, with `45 min` minimum and up to `75 min` if extra time does not create leg soreness

Movement slots:
- squat pattern
- hinge pattern
- pull pattern
- single-leg pattern
- trunk anti-rotation

Suggested exercise choices:
- goblet squat or dumbbell squat
- Romanian deadlift with dumbbells or barbell
- seated row or lat pulldown
- split squat
- Pallof press

If Centr offers different names:
- choose the closest dumbbell, machine, or cable equivalent
- preserve the movement pattern and effort target first

Recommended Centr starting points:
- `Strength for Sports`
- `Multi-Gym Strength`
- `3-Day Strength Training`
- `Power Pro`, especially lower-body strength days such as `Power Pro: Week 1, Day 3` when the session resembles squat/hinge/single-leg/core work

Verified Centr exercise links:
- Tutorial - Cable Romanian deadlift: `https://centr.com/article/show/32423/power-shred-tutorial-cable-romanian-deadlift`
- Dumbbell leg workout article, including dumbbell Romanian deadlift: `https://centr.com/blog/show/27279/dumbbell-leg-workouts`

How to choose:
- prefer a full-body or lower-body-plus-pull session over a bodybuilding split;
- choose sessions with squat, hinge, row/pull, single-leg, and trunk work;
- avoid chest/arms-only, glute hypertrophy, HIIT finishers, and hard conditioning blocks;
- if the Centr session includes too many exercises, keep the coach plan's first 4-5 movement slots and skip the rest.

Current best fit:
- Use Centr as the exercise demo/source for `Multi-Gym Strength`, `Strength for Sports`, or a lower-body `Power Pro` day, but execute the coach-prescribed table.
- If choosing a named Centr workout, prefer `Lifting: Lower Body` or `Power Pro: Week 1, Day 3` style sessions and keep intensity at RPE 5-6.
- If Centr offers an exact session with leg press or squat, Romanian deadlift, row or lat pulldown, split squat or step-up, and core, use that as the closest match.
- Keep all working sets at RPE 5-6 with 2-3 reps in reserve.

Movement slots:

- mobility finisher

Suggested exercise choices:

- dumbbell bench or machine press
- cable row
- dead bug
- side plank

Recommended Centr link:
- Centr Align: Pilates Day 1: `https://centr.com/workout/overview/0/14536/align-pilates-1-sylvia-with-tahl`

## Rules

- Use the simplest Centr equivalent that matches the planned pattern.
- Prefer an existing local exercise-library entry before introducing a new Centr option.
- Use exact verified Centr links when recommending a Centr meal, snack, workout, or exercise.
- Do not give generic Centr search keywords; if no exact link is verified, give the coach-prescribed exercise directly.
- Prefer exercises you already know how to perform well.
- Do not replace a moderate support session with a hypertrophy or max-effort workout.
- If Centr offers only a harder variation, reduce load or set count.
- If Centr gives a daily workout that conflicts with the coach plan, follow the coach plan and use Centr only for exercise substitutions.
- For meals and snacks, use Centr options that support the workout: carbs before longer rides, protein after gym, and simple recovery meals after late sessions.
