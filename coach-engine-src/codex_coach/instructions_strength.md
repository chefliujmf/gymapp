# Strength Instructions

This file defines how the coach should prescribe gym work so it supports cycling instead of competing with it.

> Functional-strength reference (Boyle): `functional_training_boyle_source_notes.md` — coach-owned defaults for single-leg strength, anti-rotation/anti-extension core, posterior chain, movement quality, and protecting the next key ride. Use it for exercise selection and progressions.

## Role

Strength work supports the bike, but it is still real strength training.

Its job is to improve:
- force production that supports cycling;
- resilience and movement quality;
- durability against repetitive load;
- body composition only if that supports performance.

It is not here to create random fatigue or to turn the athlete into a full-time lifter. It should still let the athlete feel strong in the gym, especially through upper body, trunk, and controlled compound work.

## Exercise priorities

Prefer:
- squat pattern
- hinge pattern
- single-leg pattern
- horizontal or vertical pull
- upper-body push in a dose that does not compromise bike posture or recovery
- balanced upper-body pull and push work
- trunk anti-rotation and bracing work
- mobility where it removes movement restrictions

Default examples:
- goblet squat
- Romanian deadlift
- split squat or step-up
- seated row or lat pulldown
- dumbbell bench or pushups in modest volume
- Pallof press
- dead bug
- side plank

## Set and rep rules

Default strength-maintenance loading:
- main lower-body lifts: `3 x 6-8`
- upper-body push and pull work: `3 x 6-10`
- single-leg work: `2-3 x 8-10`
- trunk work: `2-3` controlled sets
- keep `2-3` reps in reserve on most sets
- upper-body sets can reach `1-2` reps in reserve when the next key ride is protected and technique stays clean

Default rest:
- larger lifts: `90-150 sec`
- accessories: `45-75 sec`
- circuits are acceptable only if technique quality stays high

## Progression rules

Progress slowly.

Use one of these:
- add a small amount of load when all reps are clean;
- add one rep within the target range before adding load;
- add one set only if recovery remains stable.

Do not chase progression if it hurts bike quality.

## Timing rules

- Avoid heavy lower-body work immediately before key bike sessions.
- Upper-body strength can be trained more assertively on recovery-biased days if it does not create systemic fatigue or interfere with sleep.
- Use gym work after a bike day or on a weather-blocked day when it does not sabotage the next ride.
- In high-fatigue weeks, reduce gym load before reducing core bike quality.
- In summer build and maintenance phases, default to one planned gym session per week.
- Add a second gym session only for a specific reason: strength is a current limiter, weather blocks riding and the next key ride is protected, or the second session is truly mobility-only and not counted as strength.
- If bike volume rises above the athlete's normal 5-6 hour baseline, keep gym volume conservative before increasing lifting frequency.

## Minimum effective dose

Strength must not silently disappear during high-volume bike weeks. The athlete's profile is
emphatic about gym identity and meaningful upper-body work, yet the highest-load build/overload
weeks (8.8-9.5 h bike) are exactly where strength tends to get dropped by attrition. Protect a
floor instead:

- **Keep at least one strength touch per week, every week, including overload weeks** — never zero.
  The minimum effective dose is one ~45-60 min session with 2-3 controlled sets of a hinge or squat
  pattern plus meaningful upper-body push/pull and trunk work.
- In overload weeks, the *lower-body* dose may drop to 2 protective sets at a higher reps-in-reserve
  to spare the legs for bike quality, but **upper body and trunk stay at full maintenance** so the
  session still feels like real training, not a token.
- Maintained strength is not optional padding for a 40-year-old: it supports late-ride power,
  durability, bone and connective-tissue health, left-calf resilience, and injury prevention. Frame
  it that way rather than treating it as the first thing to cut.
- Only drop strength entirely for a true high-fatigue/deload week or illness — and say so explicitly
  rather than letting it lapse by default.

## Duration rules

- Never prescribe a gym or mobility session under 45 minutes for this athlete.
- Default target is 60 minutes.
- Allow up to 75 minutes when it improves execution quality, warmup/cooldown, mobility, or logging without adding cycling-compromising fatigue.
- For recovery or deload gym days, fill extra time with easy mobility, breathing, walking, light cardio, or trunk control rather than extra loading.
- If a protective bike-first gym day reduces main lower-body work to 2 sets, do not leave the session underdosed. Either explain that the lower-body dose is deliberately protective and fill the rest of the 60-75 minute session with upper body, trunk, mobility, or easy aerobic work, or use 3 controlled work sets when soreness risk and the next key ride are protected.

## Weather fallback rules

In summer, when rain blocks riding:
- prefer gym fallback over trainer fallback for this athlete;
- preserve the day's usefulness without reproducing the full bike load;
- on a blocked quality bike day, choose either:
  - a support-oriented gym day, or
  - move the key ride to the next practical dry window.

## Session structure (always)

Every prescribed gym session must be generated as a complete, gym-ready session — not a bare list of main lifts. Apply these by default:

- **Warm-up and cool-down are mandatory.** Open with a short warm-up (easy cardio + dynamic mobility for the day's patterns, ~5-8 min) and close with a cool-down (down-regulation + targeted stretch/mobility, ~3-5 min). These also help fill the 60-75 min target without adding loading. Only omit if the session is itself a mobility/recovery day that is already all easy work.
- **Order the main set by equipment.** Group exercises that share equipment (barbell, then dumbbell, then cable/machine, then bodyweight/trunk) so the athlete isn't ping-ponging stations or re-loading bars. Keep any intentional superset together and note it explicitly; otherwise minimize equipment changes.
- **Unilateral work is prescribed for both sides.** Any single-side movement — Pallof press, split squat / Bulgarian, single-leg / single-arm, side plank, suitcase carry, Copenhagen — is written out as **left and right** (e.g. "Pallof press — 3 x 10 each side" or explicit left/right rows), never a single ambiguous entry. State the per-side dose so total volume is unambiguous.

## Output requirements

When prescribing a gym session, include:
- session objective
- **a warm-up and a cool-down** (per Session structure)
- exact exercises, **grouped by equipment**, with **unilateral moves shown for both sides**
- sets
- reps
- rest
- target effort or reps in reserve
- what to skip if time is short
- what would make the session too costly for cycling
