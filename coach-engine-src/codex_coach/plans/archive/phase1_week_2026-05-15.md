## Weekly objective

Re-enter structured work with one true threshold session, one durability-focused aerobic ride, and one secondary sustained-power session without turning the week into a fatigue trap.

- Recent load on 2026-05-09 through 2026-05-11 was already meaningful.

- The point of Week 1 is not testing. The point is clean execution and good signal.

## Adaptive rules

- If the 2026-05-16 threshold session is poor, reduce the 2026-05-20 session to easy endurance.
- If the 2026-05-17 long ride leaves heavy residual fatigue, shorten the 2026-05-20 main set.
- If rain blocks the 2026-05-17 ride, use the gym fallback instead of the trainer.
- If rain blocks the 2026-05-20 ride, either move it to the next dry outdoor window or replace that day with the planned gym session and reassess.
- If life cuts the week short, keep 2026-05-16 first, then 2026-05-17, then 2026-05-20.
- Do not make up missed sessions.
